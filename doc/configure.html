Настройка работы программы

    Для работы ядра необходимо, чтобы в рабочей папке находился файл config.ini и файл базы данных. Файл конфигурации является текстовым, последовательность параметров строго определенная, пробелы недопустимы. База данных может быть создана или пересоздана с помощью кнопки Recreate DataBase. При этом пересоздается база данных и все ее таблицы начиная от списка портов и их параметров, заканчивая всеми остальными таблицами (пакеты, сообщения, станции и т. д.).

    В основных параметрах станции указывается ее позывной, координаты и Unproto Address по умолчанию. Эти данные будут подставляться в параметры вновь создаваемых портов по умолчанию. Для сохранения этих параметров в файл config.ini нужно нажать кнопку Save.

    Для начала работы ядра необходимо соединиться с системной базой данных, нажав кнопку Connect. После этого необходимо перейти на закладку портов (Ports).

    На закладке списка портов ведется настройка списка и параметра портов. Общие параметры всех портов отображаются в таблице: тип порта, его описание, позывной порта, текст маяка, интервал маяка, Unproto адрес, координаты и символ. Таким образом, имеется возможность указать уникальные параметры для каждого порта.

    При редактировании параметров порта открывается отдельное окно, в котором редактируются как основные параметры, так и специфичные для данного типа порта.

    После настройки параметров всех портов необходимо загрузить их в ядро. Для этого нажимается кнопка Up All Ports. При этом создаются соответствующие порты и начинается их работа: происходит прием, передача пакетов. Системная информация о работе портов отображается на закладке SysLog, например сообщения об ошибках при соединении с ядром agw или с Internet APRS сервером.

    Все принятые и переданные ядром пакеты отображаются на закладке Packets, сообщения на закладке Messages.
